从零搭一个 GitHub Pages 个人博客

一个好的个人博客 demo 不需要一上来就引入复杂框架。先把首页、文章列表、阅读体验和部署路径做扎实， 后续再补 Markdown 管线、评论、订阅和主题切换，会更稳。

当前版本采用纯静态文件结构，适合直接放到 GitHub Pages。视觉上用一张自定义封面图建立识别度， 内容区则保持克制，方便以后替换为真实文章。
