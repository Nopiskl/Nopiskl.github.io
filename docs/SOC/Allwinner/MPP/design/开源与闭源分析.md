## 全志 eyesee-mpp 闭源组件分析报告

经过对整个 `external/eyesee-mpp` 目录的系统性分析，以下是详细的闭源（预编译二进制库）清单：

---

### 一、视频编解码库（libcedarc） — **完全闭源**

位于 `middleware/sun8iw21/media/LIBRARY/libcedarc/library/glibc|musl/`

| 库文件 | 说明 |
|--------|------|
| `libVE.so / .a` | Video Engine 硬件加速核心驱动封装 |
| `libvideoengine.so / .a` | 视频引擎调度层 |
| `libawh264.so / .a` | H.264 硬件解码器 |
| `libawh265.so / .a` | H.265/HEVC 硬件解码器 |
| `libawmjpeg.so / .a` | MJPEG 硬件解码器 |
| `libvenc_h264.so / .a` | H.264 硬件编码器 |
| `libvenc_h265.so / .a` | H.265 硬件编码器 |
| `libvenc_jpeg.so / .a` | JPEG 硬件编码器 |
| `libvenc_common.so / .a` | 编码公共基础库 |

> 均提供 glibc 和 musl 两个版本，无源代码。

### 十、开源部分（有源代码）

相比之下，以下部分**是开源的**：

- `middleware/sun8iw21/media/` 下所有 `.c` 文件（MPI 接口层，如 `mpi_vi.c`、`mpi_venc.c` 等）
- `framework/sun8iw21/` 下的应用框架层（camera、recorder、player 等）
- `system/public/` 下的公共系统库（tinyalsa、liblog、libion 等）
- `system/private/rtsp/`（基于 live555 的 RTSP，有完整源码）
- `system/private/onvif/`（ONVIF 有源码）
- `external/` 下的第三方开源库（SQLiteCpp、jsoncpp、lz4、civetweb 等）
- `middleware/sun8iw21/media/LIBRARY/libFsWriter/`（有 .c 源码）
- `middleware/sun8iw21/media/LIBRARY/libResample/`（有 .c 源码）
- `middleware/sun8iw21/media/LIBRARY/libmuxer/`（各格式 muxer 有源码）

---

### 总结

全志 MPP 中**闭源的核心部分**集中在：
1. **硬件加速的视频编解码**（libcedarc/libVE）—— 涉及硬件 IP 授权
2. **ISP 图像算法**（全套 AE/AWB/AF 等）—— 核心竞争力
3. **AI 推理算法**（lib_aw_ai_*）—— 商业价值高
4. **音频编解码算法**（涉及专利授权）
5. **特殊算法**（EIS/ISE/ADAS）—— 差异化功能
6. **第三方 SDK**（TUTK P2P）—— 合同约束
