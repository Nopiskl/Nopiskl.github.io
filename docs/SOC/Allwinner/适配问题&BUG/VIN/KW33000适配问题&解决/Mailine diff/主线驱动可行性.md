```
  ↓
sensor_init() 返回错误（I2C 读失败）
  ↓
pipeline 半初始化，scaler 未经 S_FMT 初始化
  ↓
后续 G_SELECTION 调用 scaler 时，scaler 状态不完整
  ↓
scaler get_selection 返回 -EINVAL
```

**这不是 sensor 驱动的问题，是 scaler 驱动的问题**。

#### 4. **为什么 gc2053 没有这个问题？**

gc2053 的 `sensor_init()` 成功了，所以 pipeline 完整初始化，scaler 状态正常。

```c
// gc2053_mipi.c
static int sensor_init(struct v4l2_subdev *sd, u32 val)
{
    // gc2053 的 chip ID 读取成功
    // 所以 sensor_init() 返回 0
    // pipeline 继续初始化，scaler 状态完整
    return 0;
}
```
